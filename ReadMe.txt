You can insert data using following api's:
->To insert Student use http://localhost:4444/api/students
->To insert course use http://localhost:4444/api/courses
->To insert subjects use http://localhost:4444/api/subjects
->To insert teacher use http://localhost:4444/api/teachers
->To insert Batch use http://localhost:4444/api/studentBatchMapping/addBatch
->To insert Lecture use http://localhost:4444/api/studentBatchMapping/addLecture
->To enroll student in batch use http://localhost:4444/api/studentBatchMapping

Note: studentBatchMapping.ts is made just to insert data in Batch,Lecture,Mapping table.
